Shopify Odoo Connector

Introducing Shopify Connector in version 14 community. The module helps in connecting odoo with shopify. Sale Orders, Customers and Products can be imported/exported between odoo and shopify. Also helps to Confirm the shopify orders from odoo itself.

Configuration

Create an account in shopify and get the credentials for integration.

  • Install the module and go to shopify configuration page.
  • Create shopify instance.
  • Provide the credentials and connect.
Features

Connects Odoo with Shopify using API and Password. Products,orders and customers can be synchronized between odoo and Shopify in a button click.
